## How to Use

1. Go through each topic file
2. Read the question, try to answer before looking at the answer
3. Focus on understanding concepts, not memorization
4. Practice explaining answers out loud
5. Mark cards you struggle with for review

## Tips

- Start with your strongest topics to build confidence
- Spend more time on weak areas
- Practice with a timer (2-3 minutes per answer)
- Record yourself to check clarity
- Use these cards as a baseline, but also research company-specific questions
